Revive Book

Configuration for the Revive Book item.

Template

reviveBook:
  enabled: true
  name: "&c&lRevive Book"
  material: "ENCHANTED_BOOK"
  inventory-title: "&a&lRevive player!"
  lore:
    - "&6Revives targeted player!"
  skull-lore:
    - "&aPress on the head to revive."
  unbreakable: false

enabled

Defines whether the Revive Book item is enabled. If set to false, the item cannot be used.

Type Boolean
Default true

name

The display name of the Revive Book item. Supports color codes with & and hex colors with &#rrggbb.

Type String
Default &c&lRevive Book

material

The material of the Revive Book item. Must be a valid material from the Spigot documentation.

Type Material
Default ENCHANTED_BOOK

inventory-title

The title of the inventory that opens when using the Revive Book, showing all banned players as skulls to choose from.

Type String
Default &a&lRevive player!

lore

The lore (description) shown on the Revive Book item. Supports color codes.

Type String List
Default ["&6Revives targeted player!"]

skull-lore

The lore shown on each player skull inside the Revive Book inventory.

Type String List
Default ["&aPress on the head to revive."]

unbreakable

If set to true, the Revive Book item will not be consumed after reviving a player.

Type Boolean
Default false
