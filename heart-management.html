Heart Management

Core settings for how hearts are gained and lost.

removeHeartOnlyIfKilledByPlayer

If set to true, the player will only lose a heart when killed by another player. Deaths from mobs, fall damage, lava, etc. will not remove hearts.

If set to false, the player will lose a heart on any death.

Type Boolean
Default false

killHeartLimit

The maximum amount of health points a player can reach by killing other players. Set to 0 to disable the limit.

Type Integer
Default 0

reviveHeartAmount

The amount of health points a player will have after being revived using the Revive Book.

Type Integer
Default 20 (10 hearts)
