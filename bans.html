Bans

Settings for banning players who lose all their hearts.

Template

banOn0Hearts: true
broadcastBanFrom0Hearts: true
banTime: 0

banOn0Hearts

If set to true, players will be banned (kicked and prevented from joining) when they reach 0 hearts.

Type Boolean
Default true

broadcastBanFrom0Hearts

If set to true, when a player is banned for having 0 hearts, a broadcast message will be sent to all online players.

Type Boolean
Default true

banTime

The duration (in minutes) that a player will be banned for losing all hearts. Set to 0 for a permanent ban (until manually unbanned or revived).

Type Integer (minutes)
Default 0 (permanent)
