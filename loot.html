Loot

Configure natural loot generation in chests and minecarts across different worlds.

Template

loot:
  enabled: true
  worlds:
    world:
      chanceForHeartToGenerate: 50
      heartAddChance: 50
      chanceForReviveBookToGenerate: 50
    world_nether:
      chanceForHeartToGenerate: 75
      heartAddChance: 75
      chanceForReviveBookToGenerate: 75
    world_the_end:
      chanceForHeartToGenerate: 100
      heartAddChance: 100
      chanceForReviveBookToGenerate: 100

enabled

Defines whether the loot generation system is enabled. If set to false, no P-LifeSteal items will spawn in chests or minecarts.

Type Boolean
Default true

worlds

A list of worlds where loot generation is active. Each world has its own chance settings. Add or remove world entries to control which worlds get loot.

Type Section
Default worlds world, world_nether, world_the_end

chanceForHeartToGenerate

The chance (in %) that an Extra Heart item will be placed into a chest or minecart when a chunk is populated.

Type Integer (0-100)
Default 50 (overworld) · 75 (nether) · 100 (end)

heartAddChance

The success chance (in %) of the Extra Heart item found as loot. This is the chance that using the loot heart will actually add a heart to the player.

Type Integer (0-100)
Default 50 (overworld) · 75 (nether) · 100 (end)

chanceForReviveBookToGenerate

The chance (in %) that a Revive Book will be placed into a chest or minecart when a chunk is populated.

Type Integer (0-100)
Default 50 (overworld) · 75 (nether) · 100 (end)
