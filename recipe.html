Recipe

Configure crafting recipes for the Extra Heart and Revive Book items.

Default Recipes

By default, P-LifeSteal comes with two recipes:

Extra Heart (Recipe 1)

A shaped crafting recipe using 8 Diamond Blocks surrounding a Nether Star.

| Diamond Block | Diamond Block | Diamond Block |
| Diamond Block | Nether Star   | Diamond Block |
| Diamond Block | Diamond Block | Diamond Block |

Revive Book (Recipe 2)

A shaped crafting recipe using 6 Totems of Undying, 2 Enchanted Golden Apples, and a Book.

| Totem of Undying       | Totem of Undying       | Totem of Undying       |
| Enchanted Golden Apple | Book                   | Enchanted Golden Apple |
| Totem of Undying       | Totem of Undying       | Totem of Undying       |

Template

recipe:
  enabled: true
  recipes:
    1:
      recipe-enabled: true
      item: "extra_heart"
      shaped: true
      extraHeartItemUseSuccess: 100
      discover: false
      items:
        - "DIAMOND_BLOCK"
        - "DIAMOND_BLOCK"
        - "DIAMOND_BLOCK"
        - "DIAMOND_BLOCK"
        - "NETHER_STAR"
        - "DIAMOND_BLOCK"
        - "DIAMOND_BLOCK"
        - "DIAMOND_BLOCK"
        - "DIAMOND_BLOCK"
    2:
      recipe-enabled: true
      item: "revive_book"
      shaped: true
      discover: true
      items:
        - "TOTEM_OF_UNDYING"
        - "TOTEM_OF_UNDYING"
        - "TOTEM_OF_UNDYING"
        - "ENCHANTED_GOLDEN_APPLE"
        - "BOOK"
        - "ENCHANTED_GOLDEN_APPLE"
        - "TOTEM_OF_UNDYING"
        - "TOTEM_OF_UNDYING"
        - "TOTEM_OF_UNDYING"

enabled

Defines whether the crafting recipe system is enabled. If set to false, no custom recipes will be registered.

Type Boolean
Default true

item

The item that will be crafted by this recipe. Must be either extra_heart or revive_book.

Type String
Options extra_heart · revive_book

recipes

A numbered list of recipe definitions. You can add as many as you want by adding new numbered entries (e.g. 3:, 4:, etc.).

recipe-enabled

Defines whether this specific recipe is enabled. Allows you to disable individual recipes without removing them from the config.

Type Boolean
Default true

shaped

If set to true, the recipe is shaped (items must be placed in exact positions). If false, the recipe is shapeless (items can be placed anywhere in the crafting grid).

Type Boolean
Default true

discover

If set to true, the recipe will be automatically discovered when a player joins the server (shown in the recipe book).

Type Boolean
Default false

extraHeartItemUseSuccess

The success chance (in %) of the Extra Heart item crafted by this recipe. Only applies when item is extra_heart.

Type Integer (0-100)
Default 100

items

A list of 9 materials that define the crafting grid. For shaped recipes, items are placed in reading order (left-to-right, top-to-bottom). Materials must be valid entries from the Spigot documentation.

Type Material List (9 items) 
Crafting Grid Positions:
| A (1) | B (2) | C (3) |
| D (4) | E (5) | F (6) |
| G (7) | H (8) | I (9) |
